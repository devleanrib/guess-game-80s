# 🎮 Guess Game 80s

Projeto de jogo de adivinhação com interface retrô (anos 80), desenvolvido com foco na prática de lógica de programação e interação com o usuário.

## 🚀 Tecnologias
- HTML
- CSS
- JavaScript
- Python (lógica e estrutura do algoritmo)

## 🎯 Funcionalidades
- Geração de número aleatório
- Validação de entrada do usuário
- Feedback dinâmico (número maior ou menor)
- Animações visuais a cada interação
- Contador de tentativas
- Reset do jogo sem recarregar a página

## 💡 Objetivo
Aplicar conceitos fundamentais de lógica de programação, controle de fluxo e manipulação do DOM, com foco em construção de interfaces interativas e comportamento orientado ao usuário.

## 🧠 Aprendizados
- Estruturação de lógica condicional e loops
- Tratamento de entrada de dados
- Manipulação dinâmica de elementos com JavaScript
- Organização de código e separação de responsabilidades
- Noções iniciais aplicadas à automação e testes
